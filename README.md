# Projeto Responsividades 📐

## 📱 Sobre o Projeto
Landing page responsiva desenvolvida para simular uma loja de e-commerce mobile.
O objetivo foi praticar responsividade, HTML semântico e CSS moderno. Desenvolvido durante meus estudos no **DevClub**.
